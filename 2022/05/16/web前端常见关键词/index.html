  51. HCI（Human Computer Interaction）人机交互
  52. API（Application Programming Interface） 应用程序编程接口

  54. RGB（Red Green Blue）红色 绿色 蓝色
  55. DNS（Domain Name System）域名系统
  56. RFC（Request For Comments）协议或标准的请求注解
  57. DES（Data Encryption Standard）数据加密标准
  58. DPI（Dots Per Inch）每英寸点数
  59. GIF（Graphics Interchange Format）图形交换格式

  73. MPEG（Moving Picture Expert Group）运动图像专家组
  74. MMVF（Multi-Media Video File）多媒体视频文件
  75. MIDI（Musical Instrument Digital Interface）乐器数字接口
  76. EDIF（Electronic Design Interchange Format）电子设计交换格式
  77. JSON（JavaScript Object Notation) JS 对象简谱
  78. BSON（Binary JSON) 二进制JSON
  79. ODBC（Open Database Connectivity) 开放数据库连接
  80. JDBC（Java DataBase Connectivity) java数据库连接

  83. WCAG（Web Content Accessibility Guidelines）web内容无障碍指南
  84. CSRF（Cross Site Request Forgery）跨站请求伪造
  85. CORS（Cross-Origin Resource Sharing）跨域资源共享
  86. HREF（Hypertext Reference）超文本引用
  87. MIME (Multipurpose Internet Mail Extensions) 多用途互联网邮件扩展
  88. REPL（Read-Eval-Print Loop）读取-求值-输出的循环 交互式解释器
  89. ANSI（American National Standards Institute）美国国家标准学会
  90. CNPM（Chinese Node Package Manager）淘宝镜像Node.js包管理器
  91. JSONP（JSON with Padding）填充式JSON或参数式JSON

  94. UTF-8（8-bit Unicode Transformation Format）8位Unicode转换格式
